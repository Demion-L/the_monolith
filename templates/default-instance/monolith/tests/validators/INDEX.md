# Validator Specifications

Cognitive integrity validators that check consistency of a MONOLITH instance.

| Validator               | What It Checks                                     |
| ----------------------- | -------------------------------------------------- |
| routing-integrity.md    | routing-map tables are complete and consistent     |
| protocol-references.md  | all referenced protocols exist                     |
| mode-consistency.md     | cognitive modes are complete (7 modes, all fields) |
| privacy-routing.md      | privacy routing rules are complete                 |
| delegation-semantics.md | model delegation is correctly defined              |
